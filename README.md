# server-setup

## How to use

1. Download the script:
```
curl -s https://raw.githubusercontent.com/blumewas/server-setup/refs/heads/main/deploy-server.sh > deploy-server.sh
```

2. Make the script executable:
```sh
chmod +x deploy-server.sh
```

3. Run the script
```sh
./deploy-server.sh -u "username" -k "public_key"
```
